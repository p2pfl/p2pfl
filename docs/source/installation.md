# Installation

``` {warning}
   PyTorch version 1.11.0 is required.
```

## Install using pip

Stable releases are available on [PyPI](https://pypi.org/project/p2pfl/):

```bash
pip install p2pfl
```

## Create a virtual environment

```bash
pipenv install p2pfl
```
